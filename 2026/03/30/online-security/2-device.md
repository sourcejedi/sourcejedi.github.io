# Security part 2 \- Your device

# Lock your portable device

You should have a screen lock on your portable device.  Most people have already memorized a PIN.

If you lose the lock code for your smartphone, you will have to delete everything on your phone, and start again.

You may write down the code, in a safe place.  This is also useful for a loved one to access your device, when you are unable to do so.

This is how smartphones work.  If you are used to a computer that works differently, you can still use that type of computer\!

### Option: Unlock using your fingerprint, face, etc

Add a secure “biometric” to your screen lock.  Many devices can scan your fingerprint or face.  This makes it easier to use the screen lock, especially on mobile phones.

This does not replace the lock code for your phone.  Your phone still requires your lock code (or pattern) once every three days.  This is to test you can remember the code, in case there is a problem recognizing your face or finger.  This applies to both Android phones and iPhones.  If you cannot enter your lock code, you will lose access.

# Install your security updates

Be Cyber Aware: [Install the latest software and app updates](https://www.ncsc.gov.uk/collection/top-tips-for-staying-secure-online/install-the-latest-software-and-app-updates).

Out-of-date apps and devices will miss out on security fixes.

If your device does not have security updates, criminals could hijack your device and accounts.  New attacks are discovered every day.  Many attacks are delivered through online adverts and spam.  These risks vary depending on your usage, but you cannot avoid them entirely.

# Appendix: Traditional alternatives {#appendix:-traditional-alternatives}

If your fingers have memorized the screen lock code for your device, you can ignore this section.

If you are not used to using a securely encrypted device, alternative technologies are still available.  They are not called iPhone, or Android :-).  If you are used to computers, you can still use a computer.

By default, you can reset the PIN for a Windows computer, if you have a mobile number linked to your Microsoft Account.  Be aware:

* If your Microsoft Account has *two-step verification,* you will also need the online password for your Microsoft Account.  This is different from your Windows PIN.

* When you reset your PIN, you will lose any *passkey* saved in the Windows system (Windows Hello).

You can still buy your mobile number through a phone shop, and manage it with them.  You can still keep a password book.  You can still use a reliable black-and-white laser printer, and print out paper recovery codes (although features will vary, for different accounts).

You can still use a backup drive with Windows File History, to copy your files safely.  It still does not copy passwords, sign-ins, passkeys, etc.

Remember you can also use a social backup \- a recovery contact.  Notes:  1\) Microsoft’s design for *passkeys* (as opposed to passwords) will not be decrypted by by a “recovery email address” or “recovery phone number”.  2\) Google Password Manager apparently cannot be decrypted using a recovery contact, *if it has been encrypted*.  3\) Apple Passwords *can* be decrypted using a recovery contact.  4\) Other password managers with a feature called a “recovery contact” should work as advertised.

It is still possible to turn off encryption in Windows, and/or use an unencrypted backup drive.  It is still possible to use Windows without a password or PIN, after you disable encryption.  If you disable the password but the encryption is still enabled, then you are still at risk of losing access.  In this case, your mobile number / Microsoft Account will not help.
